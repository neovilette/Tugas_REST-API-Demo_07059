Endpoints Mahasiswa

Method Endpoint Deskripsi
GET /api_mahasiswa.php Ambil semua data mahasiswa
GET /api_mahasiswa.php?id=1 Ambil data mahasiswa berdasarkan id
POST /api_mahasiswa.php Tambah data mahasiswa baru
PUT /api_mahasiswa.php?id=1 Update data mahasiswa
DELETE /api_mahasiswa.php?id=1 Hapus data mahasiswa

Method Endpoint Deskripsi

GET /api_buku.php?id=1 Ambil data buku berdasarkan id
POST /api_buku.php Tambah data buku baru
PUT /api_buku.php?id=1 Update data buku
DELETE /api_buku.php?id=1 Hapus data buku
